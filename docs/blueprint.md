# **App Name**: Aromance

## Core Features:

- Product Catalog: Display perfumes in grids/lists with images, names, prices, and brief descriptions.
- Search and Filter: Enable keyword search with filters for brands, scents, prices, and categories.
- Shopping Cart: Implement a secure shopping cart with add-to-cart functionality.
- Secure Checkout: Provide a fast and secure checkout process with payment integrations.
- Rewards Program: Allow users to accumulate points for each dollar spent, unlocking discounts for future purchases.
- Customer Reviews Display: Display customer reviews to enhance credibility and assist purchase decisions. Repeated imagery of perfume bottles will create design cohesion.

## Style Guidelines:

- Primary color: Navy blue (#001F3F) for headers and accents, conveying professionalism.
- Accent color: Teal (#008080) for buttons and interactive elements, encouraging interactivity.
- Background color: Off-white (#F5F5F5) with gradients for clarity and depth.
- Font: 'Inter' (sans-serif) for a modern and neutral feel in both headings and body text.
- Use a subtle perfume bottle icon in the logo.
- Employ a vertical layout emphasizing product promotion and customer testimonials, mirroring FragranceX's clean aesthetic.
- Implement subtle hover effects (e.g., color change to teal) on links and buttons to provide feedback without overwhelming animations.
